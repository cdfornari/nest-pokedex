# Ejecutar en desarrollo

1. Clonar repo
2. Instalar dependencias
```
npm install
```
3. Instalar Nest CLI
```
npm i -g @nestjs/cli
```
4. Levantar db
```
docker-compose up -d
```

## Stack
* MongoDB
* NestJS
